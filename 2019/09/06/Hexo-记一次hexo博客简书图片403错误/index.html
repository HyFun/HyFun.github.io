Hexo|记一次hexo博客简书图片403错误

发表于 2019-09-06 | 更新于 2019-09-06 | 分类于 Hexo

最近在使用Hexo捣鼓博客，慢慢爬坑路，辛酸~

简书有点针对我！

问题描述

我使用简书作为我这个博客的图床，然后这两天心血来潮在捣鼓博客的时候，发现chrome浏览器第一次加载图片的时候加载不出来，单独打开图片地址能预览，然后再回到博客上图片就显示出来了（我知道这个是缓存）！

我一开始以为是我的chrome浏览器出了问题，还特地百度了

当然，结果是什么都没有查到！

今天又发生了图片看不了，这我就不能忍了，按F12，控制台报着403的错误。于是机智的我又再次搜了下度娘简书图片 403，终于有结果了，也找到了解决的办法！

问题解决

根本的问题是简书屏蔽了其他的网站使用图片外链。

我们可以屏蔽我们博客的referer。

我使用的是hexo-theme-next的主题，所以我就以此主题为示例

  1. 依次打开目录themes/hexo-theme-next/layout/_partials/head.swig这个文件

  2. 在这个文件中添加头<meta name="referrer" content="no-referrer"/>，如下图所示
